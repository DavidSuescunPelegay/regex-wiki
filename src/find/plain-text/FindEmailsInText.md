# Find emails in text

## Home > Find > Plain Text > Find emails in text

#### Operation: Find
#### Format: Plain Text
#### Name: Find emails in text

## Expression
```
([a-zA-Z0-9+._-]+@[a-zA-Z0-9._-]+\.[a-zA-Z0-9_-]+)
```

## Result
Returns a list of emails inside a plain text, it supports special characters (like '+').
