# Find URLs in text

## Home > Find > Plain Text > Find URLs in text

#### Operation: Find
#### Format: Plain Text
#### Name: Find URLs in text

## Expression
```
(?:(?:https?|ftp|file):\/\/|www\.|ftp\.)(?:\([-A-Z0-9+&@#\/%=~_|$?!:,.]*\)|[-A-Z0-9+&@#\/%=~_|$?!:,.])*(?:\([-A-Z0-9+&@#\/%=~_|$?!:,.]*\)|[A-Z0-9+&@#\/%=~_|$])
```

## Result
Returns a list of URLs inside a plain text.
