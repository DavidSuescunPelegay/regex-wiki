# Get values by JSON key

## Home > Get > JSON > Get values by JSON key

#### Operation: Get
#### Format: JSON
#### Name: Get values by JSON key

## Expression
```
(?:\"key-2)(?:\"\s?:\s?\")(.*)(?:\")
```

## Result
Returns the values inside a JSON key, in this example ```key-2``` it's been used.
