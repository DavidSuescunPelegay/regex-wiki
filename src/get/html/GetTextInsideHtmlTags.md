# Get text inside HTML tags

## Home > Get > HTML > Get text inside HTML tags

#### Operation: Get
#### Format: HTML
#### Name: Get text inside HTML tags

## Expression
```
(?<=<h1>).*(?=<\/h1>)
```

## Result
Returns the text contained inside ```<h1>``` tags, replace with your own HTML tag.
